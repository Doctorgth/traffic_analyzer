# traffic_analyzer
A high-performance SOCKS5 proxy tool written in Go for real-time network traffic analysis, packet slicing, and manual hex injection. Designed for protocol reverse engineering and security auditing.

# NetPacket-Injector-Go 🚀

**NetPacket-Injector-Go** — это мощная сетевая утилита на языке Go, предназначенная для перехвата, анализа и манипуляции игровым трафиком в реальном времени. Инструмент сочетает в себе полноценный SOCKS5-прокси сервер и графический интерфейс для интерактивного взаимодействия с пакетами.

## 🌟 Основные возможности

- **Интегрированный SOCKS5 Прокси**: Работает на порту `1050` с поддержкой авторизации (RFC 1928).
- **Интеллектуальная нарезка пакетов**: Автоматическое разделение TCP-потока на логические пакеты на основе заголовков длины (2-байтовый префикс).
- **Визуализация в Hex**: Отображение времени получения, длины, опкода и тела пакета в шестнадцатеричном формате.
- **Инъекция пакетов (Packet Injection)**: Возможность отправки произвольных Hex-данных клиенту "от имени сервера".
- **Функция "Smart Increment"**: Уникальная кнопка `Отправить +`, позволяющая инкрементировать байты-счетчики (UID) в пакетах снятия эффектов на лету.
- **Режим прерывания (Intercept Mode)**: Возможность ставить поток данных на паузу, записывать их в таблицу и отправлять клиенту выборочно.
- **Сохранение настроек**: Запоминание конфигурации отображения столбцов между сессиями.

## 🛠 Технологический стек

- **Язык**: [Go (Golang)](https://go.dev/) — высокая производительность и надежная работа с сетью.
- **GUI**: [Fyne v2](https://fyne.io/) — кроссплатформенный графический интерфейс с поддержкой аппаратного ускорения (GPU).
- **Сеть**: Кастомная реализация интерцептора на базе `armon/go-socks5`.

## 📦 Установка и запуск

### Предварительные требования
Убедитесь, что у вас установлен Go (версия 1.18+) и настроен C-компилятор (GCC) для работы графической библиотеки.

### Сборка из исходников
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-логин/NetPacket-Injector-Go.git
   cd NetPacket-Injector-Go
2. Установите зависимости:
go mod tidy
3. Запустите приложение
go run .
Конфигурация осуществляется через файл config.json
